# 🚗 Speeding Ticket Risk Analyzer

ML-powered Streamlit app that predicts your probability of getting a speeding ticket
based on traffic stop patterns across California freeways.

## Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the model
```bash
python train_model.py
```
This generates `model.pkl` using a `GradientBoostingClassifier` trained on
simulated traffic stop data with realistic probability distributions.

### 3. Run the app
```bash
streamlit run app.py
```

---

## Features
- **Risk gauge** — live Plotly gauge showing ticket probability
- **Risk curve** — how risk scales with speed over the limit
- **Enforcement heatmap** — folium heatmap of California freeway enforcement density
- **Construction zone** boost (1.8× multiplier)
- **Rush hour / late night** modifiers baked into the model

## Project structure
```
speeding_risk_app/
├── app.py            # Main Streamlit application
├── train_model.py    # Model training script (run once)
├── model.pkl         # Trained model (generated by train_model.py)
├── requirements.txt  # Python dependencies
└── README.md
```

## Model features
| Feature        | Description                        |
|----------------|------------------------------------|
| `speed_over`   | MPH above posted speed limit       |
| `hour`         | Hour of day (0–23)                 |
| `is_highway`   | 1 = freeway, 0 = surface street    |
| `speed_limit`  | Posted speed limit (55–75 mph)     |
